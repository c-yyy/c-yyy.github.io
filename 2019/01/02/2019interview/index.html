【#2019面试准备】知识点梳理

2019-01-02 summary
Word count: 378 | Reading time: 1min

技术

  • JS基础

    • 变量类型
      String、Number、Boolean、null、undefined、Symbol
      typeof、instanceof
      操作运算符

    • 作用域
      执行上下文、自由变量的向上查找

    • 深浅拷贝
      Object.assign()、展开运算符
      JSON.parse(JSON.stringify()),lodash(_.cloneDeep())

    • this
      new一个对象过程
      指向最后调用的那个对象
      箭头函数(默认指向其第一个父级对象)
      call(可以接受一个参数列表)、apply(只接受一个参数数组)、bind(返回的是一个函数)

    • 原型
      实例对象的proto指向其构造函数的prototype
      instanceof判断一个函数是否是一个变量的构造函数

    • 闭包
      函数父级作用域在定义时非执行时
      封装变量收敛修改权限，变量始终保存在内存中可供其他函数访问
      做函数返回值、做函数参数传递
      垃圾回收GC（标记清除、引用计数、ScavengeGC(From->To)）

    • 继承
      原型链继承，类实例指向prototype
      class类extends

  • 异步
    • 单线程
    • eventLoop（宏任务、微任务、执行栈）
    • jq-deferred -> Promise/Generator -> async/await
  • 浏览器
    • 事件机制
    • 跨域
    • 缓存
    • 渲染（repaint/reflow）
    • 路由原理hash模式、history模式
    • 性能监控

  • 框架

    • 三要素
      响应式原理Object.defineProperty、Proxy
      模版引擎（模版字符串->js对象）
      渲染render函数（diff后渲染差异）

    • React
      Redux
      Nextjs

    • Vue
      Vuex
      Nuxtjs

    • Angluar
      模版@Input()、@Output()
      ViewChild、ElementRef
      rxjs

  • 小程序
    • wepy
    • taro
    • 小游戏引擎
  • 工具自动化
    • git
    • rollup、webpack
  • Nodejs
  • 通讯
    • HTTP2
    • hybrid
  • 安全
    • XSS、CSRF
    • 密码加盐

技巧

  • 简历
  • 话术
